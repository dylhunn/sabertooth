What is Sabertooth?

Sabertooth is a lightweight and open-source chess engine written in C. It is fully UCI-compatible, for use with your favorite chess GUI. It currently plays at moderate-to-amateur strength, and improves every day!

Is it compatible with my system?

Sabertooth works with Mac OS X or any 64-bit flavor of Unix/Linux. It doesn't presently support Windows, but this can be easily fixed (the only incompatible library is POSIX threads).

How can I get Sabertooth?

Download it using the link above! Since Sabertooth is a UCI engine and doesn't include a GUI, you will need your own. I recommend XBoard, which has an excellent Mac port.

First, download the engine from the link above, and put it somewhere you'll remember. Then download the XBoard package for Mac or Linux.

After you install XBoard, open it, and select from the menu Engine->Load New First Engine. In the dialog box, point XBoard to the directory of the engine file, like this:

Be sure you have checked the "UCI" box.

What's coming next?

Sabertooth doesn't currently support en passant, which is the next major release item.
